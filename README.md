# Student Registration Status Checker

## Project Overview
This project is a simple program that checks a student's registration status based on the number of units they have registered. The same program was written in four different programming languages to show how each one works.

## What the Program Does
The program:
- asks the user to enter a student name
- asks for the number of registered units
- checks if the units are more than 7
- shows "Overload - Approval Required" if the units are above 7
- otherwise shows "Registration Accepted"
- prints a clear summary at the end with all the details

## How to Run the Programs

### Python
1. Open the python folder
2. Run the file using:
   python main.py

### Java
1. Open the java folder
2. Compile the program:
   javac Main.java
3. Run it:
   java Main

### JavaScript
1. Open the javascript folder
2. Install the required package:
   npm install prompt-sync
3. Run the program:
   node main.js

### C
1. Open the c folder
2. Compile the program:
   gcc main.c -o main
3. Run it:
   ./main

## What This Project Shows
This project shows how the same problem can be solved using different programming languages. It helps to understand:
- how input and output work in each language
- how decision-making (if statements) is written
- the differences in syntax between languages
- which languages are easier or more detailed to use

## Final Note
Even though the logic is the same, each language has its own style. This project helps in understanding those differences in a practical way.
