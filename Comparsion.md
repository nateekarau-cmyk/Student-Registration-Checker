# Comparison of Programming Languages

## Introduction
In this project, the same student registration program was written in four different programming languages: Python, Java, JavaScript, and C. Even though the task was the same, each language handled it in its own way. This shows how programming languages can differ in syntax, structure, and ease of use.

## Python
Python was the easiest to use. The code is short, simple, and easy to read. It does not need a lot of extra setup, so it is very beginner-friendly. It is a good choice for small programs like this one.

## Java
Java required more lines of code compared to Python. It is more strict, and everything has to be clearly defined, such as data types and classes. Even though it is longer, it is very organized and commonly used for large systems.

## JavaScript
JavaScript was also simple to write, similar to Python. It is mainly used for web development, but in this project, it was run using Node.js. It handles input a bit differently, but overall it is flexible and easy to understand.

## C
C was the most difficult out of the four. It requires more attention to detail, especially when dealing with input and variables. The code is less forgiving, but it is very fast and efficient. It is often used in systems programming.

## Key Differences
One of the main differences is how much code is needed. Python and JavaScript use fewer lines, while Java and C require more structure. Another difference is how strict the languages are. Java and C are more strict, while Python and JavaScript are more flexible.

## Conclusion
All four languages were able to solve the same problem, but in different ways. Python and JavaScript are easier for beginners, while Java and C require more understanding of programming concepts. This project shows that the choice of programming language can affect how a solution is written, even if the end result is the same.
